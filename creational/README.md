# 생성 (Creational) 패턴

생성 패턴은 객체 생성 메커니즘과 관련된 패턴으로, 객체의 생성과 조합을 캡슐화하여 특정 상황에 필요한 객체를 유연하게 생성할 수 있도록 돕습니다.

## 패턴 목록

### 자주 사용되는 패턴
*   [싱글턴 (Singleton)](./Singleton)
*   [팩토리 메서드 (Factory Method)](./FactoryMethod)
*   [빌더 (Builder)](./Builder)

### 기타 생성 패턴
*   추상 팩토리 (Abstract Factory)
*   프로토타입 (Prototype)
