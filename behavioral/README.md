# 행동 (Behavioral) 패턴

행동 패턴은 객체나 클래스 간의 상호작용 및 책임 분배 방법을 정의하는 패턴입니다. 객체 간의 결합도를 낮추면서 유연하고 효율적으로 협력할 수 있도록 돕습니다.

*   [템플릿 메서드 (Template Method)](./TemplateMethod)
*   [옵저버 (Observer)](./Observer)

*   책임 연쇄 (Chain of Responsibility)
*   커맨드 (Command)
*   이터레이터 (Iterator)
*   중재자 (Mediator)
*   메멘토 (Memento)
*   상태 (State)
*   방문자 (Visitor)
